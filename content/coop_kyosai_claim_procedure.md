# コープ共済 保険請求手続き

**作成日**: 2026-03-17
**期限**: 今週中
**案件**: 歯の詰め物

---

## 1. 電話連絡

**コープ共済お客様センター**: 0120-50-9431
- 受付時間: 月～金 9:00～18:00 / 土 9:00～17:00（日祝休み）
- 伝える内容:
  - 共済証書番号（手元に用意）
  - 「歯の詰め物が取れた/治療した件で共済金請求したい」
  - 治療日・通院回数
  - 治療費の金額

## 2. 必要書類

| 書類 | 入手先 | 備考 |
|------|--------|------|
| 共済金請求書 | コープ共済から郵送 | 電話後に届く |
| 診断書 or 領収書 | 歯科医院 | 領収書で済む場合が多い（電話で確認） |
| 共済証書番号 | 手元の証書 or マイページ | |
| 振込先口座情報 | 本人名義口座 | |
| 本人確認書類 | 運転免許証等のコピー | 必要な場合のみ |

## 3. 手続きフロー

```
[1] 電話: 0120-50-9431 に電話して請求の旨を伝える
     ↓
[2] 書類受領: 請求書類が郵送で届く（3-5営業日）
     ↓
[3] 書類記入: 請求書に記入 + 領収書/診断書を添付
     ↓
[4] 返送: 同封の返信用封筒で返送
     ↓
[5] 入金: 書類到着から約10-14営業日で振込
```

## 4. 注意事項

- 治療日から**3年以内**に請求が必要（時効）
- 歯科治療の場合、通院給付金が対象になる可能性あり
- 領収書原本が必要な場合があるため、コピーを取っておくこと
- 電話時に「領収書のみで可能か、診断書が必要か」を必ず確認
