# 営業プロセス根本分析: 588商談 → CRM受注1件の構造的原因と改善計画

**作成日**: 2026-03-16
**分析対象期間**: 2025年8月 - 2026年3月（約7ヶ月）
**データソース**: Lark CRM商談テーブル (588件) + 受注台帳 (116案件) + freee請求書

---

## Executive Summary（経営者向け要約）

**結論: 「営業が売れない」のではなく、「売れるチャネルに営業を投入していない」ことが根本原因。**

受注台帳の実績を分析すると、東海エアサービスは「売れていない」わけではない。受注台帳上の確定案件45件/総額1,852万円が存在する。しかしこの受注のほぼ全てが**インバウンド経由**（ミツモア、直接問合せ、東海工測経由、リピート）であり、新美・政木による**アウトバウンド営業（飛び込み・テレアポ）からの受注はCRM上わずか1件**に留まる。

つまり588件の商談活動は、受注にほとんど寄与していない。

| 指標 | 数値 |
|------|------|
| CRM商談数（アウトバウンド） | 588件 |
| CRM経由受注 | **1件**（和合コンサルタント \450万） |
| CRM経由受注率 | **0.17%** |
| 受注台帳の全受注（全チャネル） | 45件 / \1,852万 |
| インバウンド受注率（見積→受注） | **60.8%**（45/74閉鎖案件） |
| 月間固定費 | \69.3万 |
| ランウェイ | 約1ヶ月 |

## 2. 根本原因の構造分析

### 原因1: アウトバウンド営業の「出口設計」の欠如

**ヒアリングして帰ってくるだけで、次のステップが設計されていない。**

データ的裏付け:
- 商談ステージ未設定: 40.9%（直近dashboard: 388件がヒアリングのまま）
- 次アクション未設定: 43.3%
- 見積・金額入力: **1.1%**（545件中6件のみ）
- 結果入力: **0%**

営業プロセスが「訪問 → ヒアリング → CRM記録 → 終了」で完結してしまっている。「ヒアリング後に何をするか」のプロセスが存在しない。

### 原因2: ターゲット不一致 — 売れる相手に売りに行っていない

受注実績のある業種・サービスと、アウトバウンド営業のターゲットが乖離している。

**受注実績（受注台帳から）:**
| サービス | 受注件数 | 受注金額 | 平均単価 |
|---------|---------|---------|---------|
| 現場空撮 | 28件 | \12,103,860 | \432,281 |
| 空撮 | 10件 | \1,288,885 | \128,888 |
| ドローン測量 | 2件 | \4,590,000 | **\2,295,000** |
| その他 | 4件 | \425,272 | \106,318 |
| 点検 | 1件 | \113,200 | \113,200 |

**CRM商談で訴求している商材:**
| 商材 | 商談件数 |
|------|---------|
| 3次元計測 | 347件 |
| ドローン測量 | 343件 |
| 土量・在庫計測 | 292件 |
| 赤外線 | 152件 |

**矛盾:** 実際に売れているのは「現場空撮」（28件/\1,210万）なのに、営業では「3次元計測」「ドローン測量」「土量計測」を訴求している。ドローン測量は受注平均\229万と高単価だが受注2件のみ。現場空撮は平均\43万で量が出る。

### 原因3: チャネルROIの無視

**受注チャネル別実績:**

| チャネル | 受注数 | 受注率 | 受注金額 | コスト |
|---------|--------|--------|---------|-------|
| 直接問合せ | 8件 | 80% | \886万 | ほぼゼロ |
| ミツモア | 11件 | 65% | \329万 | 手数料のみ |
| リピート | 5件 | 83% | \315万 | ゼロ |
| 東海工測経由 | 7件 | 100% | \483万 | ゼロ |
| 展示会 | 2件 | 100% | \591万 | 出展費 |
| HP | 2件 | 67% | \553万 | 運営費 |
| 直営業(アウトバウンド) | 3件 | 100% | \107万 | 人件費高 |
| LP | 0件 | **0%** | \0 | 制作費+運用費 |

**発見:**
- **東海工測経由は受注率100%、金額\483万。この紹介元の深堀りが最優先。**
- ミツモアは件数最多（11件）で受注率65%。手数料を払ってでもここに注力すべき。
- 直営業（新美・政木のアウトバウンド）は3件\107万のみ。費用対効果が極めて低い。
- LP（ランディングページ）は受注ゼロ。投資回収できていない。

### 原因4: 営業人材の役割不一致

**新美のデータ:**
- 122件訪問、受注ゼロ
- ヒアリング内容は82%記入（仕事はしている）
- 見積提出1件のみ
- 飛び込み主体（35件/不在63%）
- テレアポは7件だがWarm率71%

新美は「訪問して話を聞く」ことはできるが、「提案してクロージングする」ことができていない。これは個人の能力問題というよりも、営業プロセスとして「ヒアリング後に見積を出す→クロージングする」という手順が教育・仕組み化されていないことが原因。

### 原因5: CRMデータ品質の低さが改善を阻害

| データ品質指標 | 数値 | 影響 |
|-------------|------|------|
| ステージ未設定 | 40.9% | ファネル分析不可 |
| 金額未入力 | 98.9% | パイプライン予測不可 |
| 結果未記録 | 100% | 勝率分析不可 |
| 次アクション未設定 | 43.3% | フォロー漏れ |
| 担当者不明 | 54.5% | 責任所在不明 |
| 放置91日超 | 73.0% | 死に案件の山 |

### 3-1. 業種別受注パターン

**ゼネコンが最大顧客層。**現場の定期撮影ニーズがリピートに繋がっている。

### 3-2. リピーター分析

| 和合コンサルタント | 1件 | \450万 |
| 東急建設 | 1件 | \50万 |
| 太平産業 | 1件 | \39万 |

**前田建設が6件リピート。**このような継続取引先を増やすことが安定収益の鍵。

### 3-3. 高単価案件の共通パターン

\100万超の受注案件:
1. 中部EEN: \522万（HP経由、その他）

**共通点: 全てインバウンドまたは紹介経由。アウトバウンド営業からの高額受注はゼロ。**

## 4. 改善アクションプラン

### Phase 0: 緊急対応（今週中 — 3/16-3/22）

#### A0-1. アウトバウンド営業の一時停止と役割転換

**新美の活動を「新規飛び込み」から「既存案件のフォロー + 見積作成」に切り替える。**

具体的に:
- 新規飛び込み: 週5日 → **週0日**（一時停止）
- 既存Hot/Warm案件フォロー: 週3日
- 見積作成・提出: 週2日
- 対象: CRM上のWarm以上31件 + 受注台帳のリピーター候補10社

理由: 飛び込み122件で受注ゼロの事実がある。同じことを続けても結果は変わらない。まずは既存の温かい案件から受注を作る。

#### A0-2. Hot/Warm案件の即時棚卸し（今週中に全件対応）

| 優先度 | 対象 | 件数 | アクション |
|--------|------|------|-----------|
| 1 | Hot案件（CRM） | 12件 | 國本が直接電話 or 同行 |
| 2 | 受注台帳のリピーター | 10社 | 再発注ヒアリング |
| 3 | Warm案件（CRM） | 19件 | 新美が電話 + 見積送付 |

#### A0-3. ミツモアの対応品質向上

ミツモアは受注率65%、11件受注の最大チャネル。対応速度と品質を上げる。
- 問合せ受信から30分以内の初回返信
- 見積テンプレートの即座送付
- フォロー電話を翌営業日に実施

### Phase 1: 短期改善（1ヶ月以内 — 4月中旬まで）

#### A1-1. インバウンド強化（受注率が既に高いチャネルを伸ばす）

| チャネル | 現状 | 目標アクション | 期待効果 |
|---------|------|---------------|---------|
| ミツモア | 11件受注/\329万 | 対応速度改善+レビュー促進 | +月2件受注 |
| HP/コーポレートサイト | 2件受注/\553万 | SEO+CTA改善 | +月1件問合せ |
| 東海工測紹介 | 7件/\483万 | 林直樹氏との定期連絡（月1回） | +月1件紹介 |
| リピーター | 5件/\315万 | 過去顧客への定期連絡 | +月1件再発注 |

#### A1-2. 営業プロセスの「出口」設計

全ての商談に以下の「出口」を設定する:

```
商談終了時に必ず以下のいずれかを実行する:

1. 見積を出す約束をする → 3営業日以内に送付
2. 次回訪問の日程を決める → CRMに次アクション日を入力
3. 決裁者への面談を打診する → 國本同行の日程調整
4. 「今は不要」と明確に断られる → 失注/Cold化してリサイクルへ

「ヒアリングだけして帰る」は禁止。
```

#### A1-3. 見積提出の自動化・高速化

- auto_quote_generator.py を活用し、ヒアリング内容から自動見積作成
- 標準単価表を整備（サービス別: 現場空撮\30-50万、ドローン測量\100-300万）
- 訪問翌日に概算見積をメール送付（スピードが受注率を上げる）

#### A1-4. CRMデータクリーンアップ

| 対象 | 件数 | アクション |
|------|------|-----------|
| 91日超放置案件 | 398件 | 一括「凍結」化。アクティブパイプラインから除外 |
| 営業見込みなし | 52件 | 「失注」ステージに移動 |
| ステージ未設定 | - | crm_stage_classifier.py で自動分類（実施済み） |
| 担当者不明 | 297件 | TAS-Automation分200件を実担当者に割当 |

### Phase 2: 中期改善（1-3ヶ月 — 6月まで）

#### A2-1. 新美の役割再定義

**現状:** フィールドセールス（新規開拓 = 飛び込み主体）
**提案:** インサイドセールス + フィールドフォロー

新しい役割:
1. ミツモア・HP問合せの初期対応（電話、現場確認）
2. 見積済み案件のフォロー（電話、再訪問）
3. 過去顧客への再発注ヒアリング
4. 現場撮影の立会い（顧客接点の維持）

**新規飛び込みは原則停止。**受注実績がないチャネルにリソースを投入し続ける合理性がない。

#### A2-2. ターゲット商材の転換

現状の営業トーク: 「3次元計測」「ドローン測量」「土量計測」
→ 受注実績に基づく転換:

| 優先度 | 商材 | 根拠 | ターゲット |
|--------|------|------|-----------|
| 1 | **現場空撮** | 28件受注、平均\43万、ゼネコンに売れる | ゼネコン現場事務所 |
| 2 | **ドローン測量** | 2件だが平均\229万の高単価 | 建設コンサル（和合型） |
| 3 | **リピート撮影** | 5件受注、コストゼロ | 既存顧客 |

#### A2-3. 紹介ネットワークの構築

東海工測経由の受注率100%は驚異的。同様の紹介元を増やす。

| 紹介元候補 | アクション |
|-----------|-----------|
| 東海工測（林直樹氏） | 月1回の情報交換。紹介フィーの設定を検討 |
| 既存顧客（前田建設、清水建設等） | 納品後に「同業他社のご紹介」を依頼 |
| 建設コンサル（ダイシン等） | 測量外注先としての位置づけを提案 |
| 建機レンタル会社 | 現場情報の共有パートナーとして提携 |

#### A2-4. 政木の活動効率化

政木は外部委託（金曜中心、週1日稼働）。限られたリソースで最大効果を出す。

- 飛び込み営業 → 停止
- 見積フォロー + クロージング電話に集中
- 三重県・岐阜県の既存リード（CRM Warm案件）への再アプローチ

### Phase 3: 長期改善（3-6ヶ月 — 9月まで）

#### A3-1. 営業チャネルの収益性評価と最適配分

月次で以下を追跡し、ROIの低いチャネルを縮小、高いチャネルを拡大:

| チャネル | KPI | 目標 |
|---------|-----|------|
| ミツモア | 月間受注数、受注率 | 月3件以上 |
| HP/SEO | 月間問合せ数 | 月5件以上 |
| 紹介 | 紹介件数、受注率 | 月2件以上 |
| リピート | 再発注件数 | 月2件以上 |
| アウトバウンド | 見積提出数、受注数 | 月3件見積、1件受注 |

#### A3-2. プライシング戦略の見直し

受注実績から判明した単価構造:
- 現場空撮: 平均\43万（ボリュームゾーン\20-60万）
- ドローン測量: 平均\229万（\100-450万のレンジ）
- 空撮: 平均\13万（低単価多数）

提案:
- 現場空撮の最低単価を\30万に設定（\10万以下の案件は受けない）
- ドローン測量はコンサル型の提案で\200万以上を狙う
- 月額パッケージ（月2回撮影で\50万/月）をゼネコン向けに提案

#### A3-3. マーケティング投資の再配分

| 現状投資 | 効果 | 提案 |
|---------|------|------|
| LP（ランディングページ） | 受注ゼロ | 停止 or 大幅改修。CTA改善 |
| Bing広告（68.6%のトラフィック） | 不明 | GA4連携でCV追跡、CPA最適化 |
| Google広告 | 不明 | GA4連携でCV追跡 |
| ミツモア | 11件受注 | プロフィール強化、レビュー獲得 |
| SEO（44記事） | HP経由2件 | 継続。CV導線強化 |

## 7. リスクと制約

| リスク | 影響 | 対策 |
|--------|------|------|
| 新美のモチベーション低下 | 飛び込み停止=「仕事を取り上げられた」と感じる可能性 | 「見積作成・クロージングという、より重要な仕事に移行」と伝える |
