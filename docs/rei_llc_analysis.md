# REI合同会社 転用検討資料

作成日: 2026-03-14
目的: REI合同会社（休眠5年）をTOMOSHI事業に転用するか、TAS部門維持かの判断材料

---

|------|------------|---------------|
| 法人格 | 合同会社（休眠5年） | TAS株式会社の一部門 |
| 設立目的 | 不動産管理 | AI×事業承継コンサル |
| 代表者 | 國本洋輔 | 同左（TAS代表） |

## 2. 合同会社の均等割コスト

資本金1,000万円以下・従業者50人以下の場合:

| 内訳 | 年額 |
|------|------|
| 都道府県民税 均等割 | 約2万円 |
| 市町村民税 均等割 | 約5万円 |
| **合計** | **約7万円/年** |

**休眠中の均等割について:**
- 正式に休眠届を提出し、事業活動・資金移動が一切ない場合、自治体によっては均等割が免除・減免される
- 名古屋市の場合は個別確認が必要
- 事業を再開すれば均等割は通常通り発生する

## 3. 休眠会社の復活手続き

### 合同会社の大きな利点
- **役員任期がないため「みなし解散」の対象外**（株式会社は12年未登記で職権解散リスク）
- 休眠5年でも法人格はそのまま維持されている

### 復活に必要な届出（費用ゼロ）

| 届出先 | 届出書類 | 費用 |
|--------|---------|------|
| 税務署 | 異動届出書（事業再開） | 無料 |
| 都道府県税事務所 | 異動届出書 | 無料 |
| 市区町村役所 | 異動届出書 | 無料 |

**所要期間:** 届出自体は即日〜数日で完了

### 追加で必要になる作業
- 休眠期間中の確定申告（未申告の場合は遡及申告が必要な可能性あり）
- 銀行口座の凍結解除（休眠により口座停止されている場合）
- 社会保険・労働保険の再加入手続き（従業員を雇う場合）

## 4. 事業目的変更手続き（不動産管理 → AI/DXコンサル）

### 手続きフロー

```
1. 総社員の同意書作成（一人社長なので自分だけでOK）
2. 定款の事業目的を書き換え
3. 法務局で目的変更登記申請
4. 税務署等へ異動届出書提出
```

### 費用

| 項目 | 費用 |
|------|------|
| 登録免許税 | **3万円** |
| 司法書士に依頼する場合 | 1〜3万円（追加） |
| 自分で申請する場合 | 3万円のみ |

**注意:** 目的変更と同時に商号変更（REI → TOMOSHI等）も行う場合、同一申請で3万円に収まる。

### 推奨する事業目的の記載例
1. AI・人工知能を活用した業務改善コンサルティング
2. 事業承継に関するコンサルティング
3. 企業のデジタルトランスフォーメーション支援
4. ソフトウェアの企画、開発、販売
5. 不動産の管理（既存目的を残すことも可能）
6. 前各号に附帯関連する一切の事業

## 5. 比較分析

### 案A: REI合同会社をTOMOSHI事業に転用

#### メリット
1. **責任分離**: TASのドローン事業とTOMOSHIのM&A/コンサル事業でリスクが分離される。一方の事業で訴訟・債務が発生しても他方に影響しない
2. **財務の透明性**: 事業別の損益が明確になり、投資判断がしやすい
3. **将来のEXIT柔軟性**: TOMOSHI事業を売却する場合、法人ごと譲渡が可能
4. **既存法人の活用**: 新規設立に比べて法人番号・銀行口座が既にある（復活するだけ）
5. **消費税の免税**: 休眠復活後、基準期間の課税売上1,000万円以下なら免税事業者として再スタートできる可能性あり（※要税理士確認）
6. **ブランド分離**: TASとTOMOSHIは既に「混ぜない」方針。法人レベルで分離することで一貫性が出る

#### デメリット
1. **維持コスト増**: 均等割 約7万円/年 + 決算申告費用（税理士費用 年10〜20万円程度）
2. **管理工数**: 2法人分の決算・申告・届出が必要
3. **社会保険の二重加入問題**: 2法人で代表を務める場合、報酬按分で社会保険料を案分する手続きが発生
4. **税務調査リスク**: 同一代表者の複数法人間取引は税務署に注視される。利益移転・価格操作と見なされないよう適正な取引が必要
5. **合同会社の信用力**: 株式会社と比較して社会的認知度が低い。特にM&A・事業承継の分野では取引先（金融機関・税理士法人等）から見劣りする可能性
6. **初期費用**: 目的変更登記3万円 + 商号変更する場合も同額内

### 案B: TAS株式会社の部門としてTOMOSHI事業を維持

#### メリット
1. **コスト最小**: 追加の均等割・決算費用なし
2. **管理シンプル**: 1法人の決算・申告のみ
3. **株式会社の信用力**: 「東海エアサービス株式会社 TOMOSHI事業部」として株式会社の看板を使える
4. **社会保険手続き不要**: 按分計算なし
5. **即座に運用可能**: 法的手続き一切不要

#### デメリット
1. **責任が混在**: ドローン事業のリスク（墜落事故・損害賠償等）がTOMOSHI事業の資産にも及ぶ
2. **損益混在**: 事業別の収益性が見えにくくなる（部門別管理で対応可能だが手間）
3. **EXIT困難**: TOMOSHI事業だけを売却する場合、事業譲渡スキームが必要（法人譲渡より複雑）
4. **ブランド矛盾**: 「東海エアサービス」の名前がTOMOSHIのM&A/AI事業と乖離

---

## 6. 法的リスク分析（責任分離の観点）

### TAS一体運営のリスクシナリオ
- **ドローン墜落事故** → 損害賠償請求がTAS全資産に及ぶ → TOMOSHI事業の資金も差押え対象
- **M&Aアドバイザリーの瑕疵** → 損害賠償がTASのドローン事業資金にも影響

### 法人分離のリスク軽減効果
- 各法人の責任は原則としてその法人の資産に限定される
- ただし**法人格否認の法理**: 法人格が形骸化している場合（資金混同、経理混同等）は分離が否定される可能性
- **対策**: 経理・口座・契約を完全に分離し、法人間取引は時価で行う

## 7. 信用力への影響

| 観点 | REI合同会社 | TAS部門 |
|------|-----------|---------|
| 対金融機関 | やや弱い（合同会社の認知度低） | 株式会社の信用力を活用可 |
| 対M&A仲介会社 | 信用構築が必要 | TASの実績を背景にできる |
| 対中小企業オーナー | 法人格の種類より実績重視 | 同左 |
| 対税理士・士業 | 合同会社でも問題なし | 同左 |
| 名刺・Web | 「TOMOSHI合同会社」は違和感なし | 「東海エアサービス TOMOSHI事業部」はやや長い |

**補足:** Apple Japan、Google Japan、Amazon Japanも合同会社。IT/DX系では合同会社は違和感がない。

## 8. 税務上の影響

### 複数法人運営時の注意点
1. **役員報酬の按分**: 2法人から報酬を受ける場合、合算で所得税・住民税が計算される（節税効果は限定的）
2. **法人間取引の適正価格**: 業務委託等を行う場合は時価で行い、議事録を残す
3. **消費税**: 各法人で基準期間の課税売上を判定。REIは復活当初は免税事業者の可能性あり
4. **法人税の軽減税率**: 各法人で年800万円以下の所得に15%の軽減税率が適用される（2法人に分散するメリットは現時点では限定的。TOMOSHI事業の売上がまだ少ないため）
5. **社会保険**: 2法人の代表の場合、主たる事業所を選択し、報酬按分で保険料を算定

### 現時点での税務インパクト
- TOMOSHI事業はまだ売上がほぼゼロの段階
- 法人分離による節税メリットは現時点では**ほぼなし**
- むしろ均等割7万円＋決算費用が純粋なコスト増

## 10. 推奨案

### 結論: **現時点では案B（TAS部門維持）を推奨。条件付きで将来の法人分離を計画する。**

### 理由

1. **ランウェイ1ヶ月の状況で年17〜27万円の固定費増は避けるべき**
2. TOMOSHI事業はまだ売上ゼロの段階。法人分離のメリット（責任分離・EXIT柔軟性）が活きるのは事業が軌道に乗ってから
3. 合同会社の信用力の低さは、M&A/事業承継分野では不利に働く可能性がある
4. REI合同会社は**解散せず休眠を継続**しておけばコストゼロで温存できる

### 法人分離に踏み切るタイミング（トリガー条件）

以下のいずれかを満たした時点で再検討:

| トリガー | 理由 |
|---------|------|
| TOMOSHI事業の月次売上が安定して50万円超 | 法人維持コストを事業収益で賄える |
| M&A案件の取扱い開始（実際に企業買収に着手） | 責任分離の実益が大きくなる |
| 外部パートナー（投資家・共同経営者）の参画 | 持分・利益配分を明確にする必要 |
| TASのドローン事業で高リスク案件が増加 | 責任分離の緊急性が高まる |

### REI合同会社の当面の扱い
- **休眠を継続**（均等割免除を自治体に確認済みであれば維持コストゼロ）
- 解散はしない（復活が容易な合同会社の利点を温存）
- 将来TOMOSHI事業が成長した際、REIを**株式会社に組織変更**してから転用する選択肢も検討可能（組織変更費用: 登録免許税3万円＋定款認証約5万円）

### 今すぐやること
1. REI合同会社の休眠届が正式に提出済みか確認
2. 均等割の免除状況を所轄自治体に確認
3. 休眠期間中の確定申告状況を確認（未申告なら税理士に相談）
