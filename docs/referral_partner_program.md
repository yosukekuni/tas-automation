# 東海エアサービス 紹介パートナー制度 設計書

作成日: 2026-03-19
対象事業: ドローン測量（tokaiair.com）
ステータス: 設計完了・実装待ち

---

## 1. 制度概要

### 目的
測量・土木・建設・不動産分野の業者・専門家が東海エアサービスの顧客を紹介する仕組みを構築し、
現状の商談→受注転換率の低さ（241件→1件）を補う新規獲得チャネルとして機能させる。

### 基本方針
- 紹介者が「紹介しやすい」シンプルな仕組みにする
- 成約ベースの報酬（紹介者にリスクなし）
- 信頼関係のある業者との相互送客を狙う

## 2. パートナー対象

### 2-1. 優先ターゲット業種

| 業種 | 紹介可能案件 | 親和性 |
|------|------------|--------|
| 測量士事務所 | 地形測量・現況測量の外注 | 最高（業務補完） |
| 土地家屋調査士事務所 | 土地境界・分筆前の現況把握 | 高（前工程） |
| 建設コンサルタント | 土量計算・施工管理支援 | 高（ICT活用） |
| 設計事務所（土木・建築） | 現地測量・3Dモデル作成 | 高（設計前工程） |
| 不動産会社（売買・開発） | 土地開発前の地形把握 | 中（案件発掘） |
| 建設会社・ゼネコン | ICT土工・出来高管理 | 高（施工段階） |
| 農業法人・土地改良区 | 農地測量・水路管理 | 中（拡大中） |
| 産廃・残土処理業者 | 容量計算・土量管理 | 中 |

### 2-2. 除外すべき相手

| 除外対象 | 理由 |
|---------|------|
| 他社ドローン測量会社 | 直接競合 |
| 測量機器販売会社（自社でドローン測量を始めた先） | 競合化リスク |
| 単発紹介のみ目的の仲介業者（紹介ブローカー） | 質の低い案件混入リスク |

## 3. インセンティブ設計

### 3-1. 紹介手数料（金銭報酬）

業界慣行として、測量・建設分野の下請け紹介手数料は **成約額の5〜10%** が相場。
信頼関係を重視しつつ、相手にメリットが伝わる水準に設定する。

| 案件規模 | 手数料率 | 上限 | 備考 |
|---------|---------|------|------|
| ～50万円 | 成約額の10% | 5万円 | 小規模案件も誘引 |
| 50〜200万円 | 成約額の8% | 16万円 | 標準案件 |
| 200万円超 | 成約額の5% | 上限なし | 大型案件 |

**支払条件:**
- 顧客からの入金確認後30日以内に振込
- 紹介者からの請求書ベース（または弊社から支払通知書発行）
- インボイス対応（適格請求書）

**有効期間:**
- 紹介連絡から **90日以内** の受注を対象（長すぎると管理困難）

### 3-2. 非金銭インセンティブ

| インセンティブ | 内容 |
|--------------|------|
| 技術情報の提供 | ICT測量・点群データの活用事例・勉強会への招待 |
| 相互送客 | 弊社顧客に対してパートナー業者を優先紹介 |
| 共同受注 | 測量士が資格要件で必要な案件での共同プロポーザル |
| データ共有 | 成果データ（点群・CAD）のフォーマット柔軟対応 |
| 優先対応 | パートナー紹介案件は見積もり優先対応（48時間以内） |

## 4. 紹介フロー

### 4-1. 受付からクロージングまで

```
[パートナー]
  顧客の測量ニーズを把握
      ↓
  東海エアサービスに連絡（メール/専用フォーム/電話）
  ※事前連絡必須（紹介元の確定のため）
      ↓
[東海エアサービス]
  CRMに商談登録（紹介元フィールドに記録）
      ↓
  顧客に初回連絡（24時間以内）
      ↓
  現地調査・見積もり
      ↓
  受注 or 失注
      ↓
[受注時]
  顧客から入金確認
      ↓
  手数料計算（CRM自動 or 手動）
      ↓
  パートナーへ支払通知（メール）
      ↓
  請求書受領
      ↓
  freee経由で振込（翌月末）
```

### 4-2. 紹介受付方法

| 方法 | 詳細 |
|------|------|
| 専用メール | partner@tokaiair.com（または既存メールに件名「[紹介]」を付ける） |
| 専用フォーム | tokaiair.com/partner/ に設置（Google Forms or CF7） |
| 電話 | 既存代表番号でも受付（担当者がCRMに手入力） |

### 4-3. 進捗報告

- 紹介受付後：受付確認メールを即日送信
- 初回面談後：商談状況をパートナーに報告（任意・要望ベース）
- 受注/失注確定時：結果連絡を必ず実施
- 手数料支払時：支払通知書をメールで送付

## 5. パートナー募集方法

### 5-1. LP設計（tokaiair.com/partner/）

**ページ構成案:**

```
1. ファーストビュー
   「ドローン測量の案件を東海エアサービスにご紹介ください」
   「成約時に成約額の5〜10%をお支払いします」

2. こんな方にお勧め
   - 測量士・土地家屋調査士（業務範囲外の案件対応に）
   - 設計事務所・建設会社（協力会社として）
   - 不動産・土木コンサル（ワンストップ提案のために）

3. 紹介手数料体系（シンプルな表）

4. 紹介の流れ（3ステップで図解）
   ①紹介連絡 → ②弊社が対応 → ③成約で報酬

5. 東海エアサービスの強み
   - 愛知・三重・岐阜・静岡対応

   - 点群・3Dモデル・土量計算まで一貫対応

```

## 6. リスク管理

### 6-1. 紹介トラブル防止

| リスク | 対策 |
|--------|------|
| 二重紹介（複数パートナーが同一顧客を紹介） | 先着優先：CRMに紹介日時を記録し、最初の紹介者に手数料支払い |
| 案件横取り（パートナーが顧客を直接対応） | 契約書の禁止事項に明記 |
| 紹介後の顧客直接アプローチ | 顧客情報の共有範囲を限定（案件概要のみ）|
| 手数料計算ミス | CRMでの自動計算・二重チェック |
| 未払いトラブル | 支払条件の書面明確化・支払通知書発行 |
| 質の低い案件の大量投入 | 成約ベース報酬のため金銭リスクなし（対応工数は要注意） |

### 6-2. 契約書ひな形の必要性

**必要性: 高（推奨）**

最初は口頭・メール合意でも運用可能だが、以下の観点から契約書（覚書）の準備を推奨する。

**最低限盛り込む条項:**

1. 契約目的・定義（紹介パートナーの役割）
2. 手数料体系・計算方法・支払条件
3. 有効期間（紹介から90日以内）
4. 情報の取り扱い（個人情報・秘密保持）
5. 二重紹介の取り扱い（先着優先ルール）
6. 禁止事項（直接営業・競合他社への同時紹介等）
7. 反社会的勢力排除条項
8. 契約解除・変更
9. 準拠法・管轄（名古屋地裁）

**当面の運用方針:**
- 初期（試験運用）: 制度案内メール＋返信承諾で運用
- 本格展開時: 覚書テンプレートを作成・締結（弁護士レビュー推奨）

### 7-2. 取引先テーブル

- 既存の `取引先種別` フィールドに「紹介パートナー」を追加
- パートナー専用のビューを作成（フィルタ: 取引先種別=紹介パートナー）

### 7-3. 自動化（将来実装）

- 受注確定時：紹介手数料を自動計算してCRMに記録
- 月次：未払い手数料レポートをメールで通知
- 90日経過アラート：未受注案件の紹介期限切れ通知

## 10. 想定Q&A（パートナー向け）

**Q. 競合他社にも紹介していいですか？**
A. 問題ありません。ただし、同一案件の二重紹介はご遠慮ください。

**Q. 紹介後にどこまで関与できますか？**
A. 商談・施工への同席は歓迎します。顧客への直接営業はお控えください。

**Q. 手数料はいつ支払われますか？**
A. 顧客からの入金確認後30日以内に振り込みます。

**Q. 見積もりだけでも手数料もらえますか？**
A. 手数料は成約時のみです。ただし、見積もり対応は無償で迅速に行います。

**Q. 紹介件数に制限はありますか？**
A. 制限はありません。多くご紹介いただけると幸いです。
