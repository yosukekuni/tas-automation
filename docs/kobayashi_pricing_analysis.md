## 3. 価格調整の選択肢

### A案: パートナー卸値設定

**概要**: 小林さんに対して、案件紹介手数料＋品質保証料という形でTASマージンを圧縮。

| 項目 | 金額 | 備考 |
|------|------|------|
| 小林さんの受注価格（エンド向け） | ¥700,000-800,000 | 現状50-60万より高く設定可能 |
| TAS取り分（紹介料+QC料） | ¥100,000-150,000/件 | 小林さんから受領 |
| 小林さん手取り | ¥550,000-700,000 | 現状より改善 |

**メリット**:
- 小林さんの営業意欲向上（価格競争力あり）
- TASは低労力で安定収入
- 案件数増加で総収入増の可能性

**デメリット**:
- TAS単価が大幅減（120万→10-15万/件）
- 品質管理の責任範囲が曖昧になるリスク
- エンドクライアントとの直接関係が薄くなる

**利益シミュレーション（年間）**:

| シナリオ | 件数 | TAS収入/件 | 年間TAS収入 |
|----------|------|-----------|------------|
| 現状（TAS主導） | 2件 | ¥300,000 | ¥600,000 |
| A案・少数 | 6件 | ¥120,000 | ¥720,000 |
| A案・拡大 | 12件 | ¥120,000 | ¥1,440,000 |

### B案: 作業分担モデル

**概要**: 撮影は小林さん、後処理（QC・納品管理・RTK測量）はTAS。役割を明確に分離。

| 工程 | 担当 | 金額 |
|------|------|------|
| 撮影 | 小林さん | ¥250,000-300,000 |
| RTK測量・位置出し | TAS | ¥80,000-100,000 |
| 補正・スティッチング | 小林さん | ¥100,000-150,000 |
| QC・コンタクトシート検証 | TAS | ¥30,000-50,000 |
| プロジェクト管理・納品 | TAS | ¥50,000-80,000 |
| **エンド向け合計** | | **¥700,000-800,000** |
| **TAS取り分** | | **¥160,000-230,000** |
| **小林さん取り分** | | **¥350,000-450,000** |

**メリット**:
- TASの技術的付加価値（RTK測量）を維持
- 品質管理がTAS側に残るため責任明確
- エンドクライアントとの関係維持

**デメリット**:
- 現場にTASスタッフ派遣が必要（RTK測量時）
- 工程間の引き継ぎコスト
- 小林さんの取り分が少なめ（単独の場合より低い可能性）

**利益シミュレーション（年間）**:

## 4. 総合比較と推奨

### 各案の比較マトリクス

| 評価軸 | A案（卸値） | B案（分担） | C案（グレード分け） |
|--------|:-----------:|:-----------:|:-------------------:|
| TAS単価 | 低 | 中 | 中-高 |
| 案件獲得力 | 高 | 中 | 高 |
| TAS工数 | 低 | 中 | 高 |
| 品質管理 | 低 | 高 | 中 |
| 小林さんメリット | 高 | 中 | 高 |
| 管理複雑度 | 低 | 中 | 高 |
| ブランド維持 | 低 | 高 | 中 |
| **年間収入ポテンシャル** | **中** | **中-高** | **高** |

### 推奨: A案 + B案のハイブリッド

**理由**:
1. **ランウェイ1ヶ月の現状**では、案件数の最大化が最優先
2. 小林さんが営業から撮影・補正まで一気通貫で動ける体制が最も効率的
3. TASはRTK測量とQCという**替えが効かない技術**で付加価値を出す

**具体的な提案**:

| パターン | 条件 | TAS取り分 |
|----------|------|-----------|
| **紹介のみ** | 小林さんが完結する案件 | 売上の15%（紹介料） |
| **QC付き** | TASブランドで納品する案件 | 売上の20% + QC費¥30,000 |
| **フル協業** | RTK測量が必要な案件 | 売上の25% + 測量費¥80,000 |

**小林さんへの提示価格例（6階建てマンション基準）**:

| パターン | エンド価格 | 小林さん | TAS |
|----------|-----------|----------|-----|

| QC付き | ¥700,000 | ¥530,000 | ¥170,000 |
| フル協業 | ¥800,000 | ¥520,000 | ¥280,000 |

## 5. 判断のためのポイント

### 確認すべき事項

1. **小林さんの案件パイプライン**: 月何件の眺望撮影案件が見込めるか
2. **小林さんの処理能力**: 月何件まで対応可能か（撮影+補正のキャパシティ）
3. **エンドクライアントの価格感度**: 60万と80万で成約率がどう変わるか
4. **RTK測量のニーズ**: エンドクライアントが位置精度をどこまで求めるか
5. **小林さんの意向**: TASとの協業形態として何を望んでいるか

### 損益分岐点

- 現状TAS主導で年2件×粗利30万 = 年60万
- ハイブリッド案で年60万を超えるには:
  - 紹介のみ: 年7件以上
  - QC付き: 年4件以上
  - フル協業: 年3件以上

**結論**: 小林さんが月1件以上の眺望撮影案件を持ち込める見通しがあれば、価格を下げてでも案件数で稼ぐ戦略が合理的。まずは小林さんに案件見込み数をヒアリングし、その上でA+Bハイブリッド案をベースに条件交渉するのが最善。
