# TOMOSHI Googleビジネスプロフィール(GBP)作成手順

作成日: 2026-03-14 / ステータス: 凍結（設計書のみ）

## 概要

TOMOSHI事業部（https://tomoshi.jp/）のGoogleビジネスプロフィールを作成し、ローカル検索での露出を確保する。

## 前提情報

- 事業名: TOMOSHI（東海エアサービス株式会社のAI活用コンサルティング事業部）
- サイト: https://tomoshi.jp/
- メール: info@tomoshi.jp
- 所在地: 名古屋市内（東海エアサービス本社住所）

## 作成手順

### Step 1: Googleアカウント準備

1. https://business.google.com/ にアクセス
2. TOMOSHI管理用のGoogleアカウントでログイン
3. 「ビジネスを追加」→「ビジネスを1件追加」

### Step 2: ビジネス情報入力

| 項目 | 設定値 |
|------|--------|
| ビジネス名 | TOMOSHI - AI活用コンサルティング |
| カテゴリ（メイン） | 「経営コンサルタント」or「ITコンサルタント」 |
| カテゴリ（追加） | 「コンピュータコンサルタント」「ビジネスコンサルタント」 |
| 住所 | 東海エアサービス本社住所 |
| サービス提供エリア | 愛知県、岐阜県、三重県、静岡県 |
| 電話番号 | TOMOSHI専用番号（or 本社番号） |
| ウェブサイト | https://tomoshi.jp/ |

### Step 3: オーナー確認

- ハガキ確認（最大14日）or 電話確認 or メール確認
- 即時確認: Search Consoleでtomoshi.jpを事前認証しておくと早い

### Step 4: プロフィール最適化

**投稿するコンテンツ:**
- サービス説明: AI導入支援、業務自動化、DXコンサルティング
- 事例紹介（社外秘に注意、匿名化必須）
- ブログ記事リンク

**写真:**
- ロゴ画像
- オフィス写真
- チーム写真（任意）

**サービスメニュー:**
- AI導入診断（無料）
- 業務自動化コンサルティング
- DX推進支援

### Step 5: 口コミ獲得戦略

1. **初期口コミ（最重要）**
   - 既存クライアントに直接依頼（面談後にフォローアップメールで）
   - 目標: 最初の3ヶ月で5件以上

2. **継続的な口コミ獲得**
   - プロジェクト完了時にレビュー依頼メール自動送信（deal_thankyou.pyに組み込み可能）
   - QRコード付き名刺・資料にレビューリンク掲載

3. **口コミ返信**
   - 全口コミに24時間以内に返信
   - ポジティブ: 感謝+具体的な振り返り
   - ネガティブ: 謝罪+改善策の提示

## カテゴリ選定の注意点

- 「AI」を含むカテゴリはGoogleに存在しない可能性がある（要確認）
- 「経営コンサルタント」が最も近い汎用カテゴリ
- カテゴリは後から変更可能

## 工数見積もり

| 作業 | 工数 |
|------|------|
| GBP作成・情報入力 | 1時間 |
| オーナー確認待ち | 1〜14日 |
| プロフィール最適化 | 2時間 |
| 口コミ依頼テンプレート作成 | 1時間 |
| **合計** | **4時間 + 確認待ち** |

## 優先順位

**低**: TOMOSHI事業部の売上が本格化してから着手。GBP自体は無料なので、余裕があれば先に作成しておくのも可。
