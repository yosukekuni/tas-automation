# ボルボ自動車保険 分析レポート

作成日: 2026-03-15

## 現状

### NASアクセス状況
- Synology DS420+ のNASマウントポイント（/mnt/nas, nas1-4）は存在するが、**現在マウントされていない（空）**
- 保険データファイルはローカルPC上にも見つからず

### NASマウント手順（要対応）

WSLからSynology NASをマウントするには以下を実行:

```bash
# 1. NASのIPアドレスを確認（通常 192.168.x.x）
# Synology DSM管理画面 または ルーターのDHCPリストで確認

# 2. cifsユーティリティをインストール（未インストールの場合）
sudo apt install cifs-utils

# 3. マウント実行（共有フォルダ名・ユーザー名は要調整）
sudo mount -t cifs //192.168.x.x/共有フォルダ名 /mnt/nas -o username=ユーザー名,password=パスワード,iocharset=utf8

# 4. 保険データを検索
find /mnt/nas -name "*保険*" -o -name "*insurance*" -o -name "*ボルボ*" -o -name "*volvo*" 2>/dev/null
```

## 保険料見直しチェックリスト（データ取得後に分析）

### 1. 現状把握項目
- [ ] 保険会社名・証券番号
- [ ] 保険期間（満期日）
- [ ] 年間保険料（月払い/年払い）
- [ ] ノンフリート等級（現在の等級・事故有係数適用期間）
- [ ] 車両情報（型式・年式・車両価額）

### 2. 補償内容確認
- [ ] 対人賠償（通常: 無制限）
- [ ] 対物賠償（通常: 無制限）
- [ ] 人身傷害（3,000万〜5,000万が一般的）
- [ ] 車両保険（一般/エコノミー/なし）
  - 免責金額（0-10万/10-10万 等）
  - 車両保険金額
- [ ] 弁護士費用特約
- [ ] ロードサービス
- [ ] 個人賠償責任特約

### 3. コスト削減の主要ポイント

#### A. 車両保険の見直し（最大削減効果）
- **ボルボの車両価額が低下している場合**: 車両保険を外す or エコノミー型に変更で年間3〜8万円削減の可能性
- **免責金額の引き上げ**: 0-10万 → 5-10万 で数千円〜1万円削減
- 目安: 車両価額100万円以下なら車両保険不要の判断もあり

#### B. 補償の重複排除
- 人身傷害と搭乗者傷害の重複
- 個人賠償責任特約が他の保険（火災保険等）と重複していないか
- ロードサービスがJAF等と重複していないか

#### C. 割引の適用確認
- インターネット割引（ダイレクト型の場合）
- ゴールド免許割引
- 証券不発行割引
- 早期契約割引（満期の45〜90日前に更新）
- 新車割引（初度登録から25ヶ月以内）

#### D. 保険会社の変更
- 代理店型 → ダイレクト型（ネット通販型）で20〜40%削減の事例多数
- ボルボ対応のダイレクト型: SBI損保、ソニー損保、チューリッヒ、アクサダイレクト等
- 注意: 輸入車は保険会社によって引受不可や割増の場合あり

### 4. ボルボ特有の注意点
- 輸入車は部品代が高いため車両保険料が割高
- 型式別料率クラスが高い傾向（特に車両・修理費クラス）
- ディーラー提携保険は割高な場合が多い
- ADAS（先進安全装備）搭載車はフロントガラス交換費用が高額（カメラ再キャリブレーション含む）

## 次のアクション

2. **データ取得後**: 上記チェックリストに沿って現状を整理

4. **判断**: 満期日に合わせて切り替え or 現状維持を決定
