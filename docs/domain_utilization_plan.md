# サービス別LP用ドメイン活用案

作成日: 2026-03-14

## 前提条件

- 本体サイト: tokaiair.com（WordPress / LiteSpeedキャッシュ）
- 未使用ドメイン: あり（tokaiair.com用に取得済み）
- 流入構成: Bing 68.6% / Google 11.5%
- ランウェイ: 1ヶ月（生存最優先フェーズ）
- コンテンツ資産: 土量コスト記事44本 + 買い手向け15本 / FAQ30問
- 営業課題: 241商談 → 受注1件（CVR改善が急務）

---

## 4案の比較

### 案1: 広告用LP専用ドメイン

**概要**: 広告出稿時のランディングページを別ドメインに置き、SEO本体と分離する。

| 項目 | 評価 |
|------|------|
| メリット | SEOドメインにnoindex/低品質ページが混在しない。A/Bテストを自由に実施可能。広告品質スコアに最適化したページ設計ができる |
| デメリット | 別サーバー/別WordPress(または静的サイト)の管理が必要。ドメインのトラストがゼロからスタート。コンバージョントラッキングがクロスドメインになり設定が複雑化 |
| SEO影響 | 本体への影響なし（分離が目的）。ただしLP側にSEO価値は蓄積されない |
| 管理コスト | 中〜高（サーバー費用+SSL+WordPress保守 or 静的ホスティング） |
| 初期構築 | 1〜2週間 |

### 案2: サービス別マイクロサイト

**概要**: ドローン測量・点群処理・DXコンサルなどサービスごとに独立サイトを構築。

| 項目 | 評価 |
|------|------|
| メリット | サービス特化のブランディング。ドメイン名にサービスキーワードを含められる（部分一致ドメインの効果は限定的） |
| デメリット | コンテンツの重複管理。被リンク・ドメインパワーが分散。複数サイトの保守負荷が大幅増加 |
| SEO影響 | **悪影響大**。既存の44本+15本の記事資産によるドメインパワーが分散。Bing/Googleともに1ドメインに集約した方が有利 |
| 管理コスト | 高（サイト数×保守コスト） |
| 初期構築 | 2〜4週間/サイト |

### 案3: 営業用短縮URL

**概要**: 別ドメインを短縮URLサービスとして使用。名刺・提案書・メール署名で使用。

| 項目 | 評価 |
|------|------|
| メリット | 即日導入可能。301リダイレクトで本体にSEO価値を集約。営業資料がスッキリする。クリック計測でどの資料が読まれているか分析可能 |
| デメリット | ドメインの活用度としては限定的。短縮URLサービスの構築/運用が必要（ただしCloudflare Workersなどで最小構成可能） |
| SEO影響 | 301リダイレクトなので本体にプラス。被リンクも本体に集約される |
| 管理コスト | **最小**（Cloudflare Workers無料枠 or 簡易リダイレクト設定のみ） |
| 初期構築 | 数時間〜1日 |

### 案4: tokaiair.comへの単純リダイレクト

**概要**: ドメイン全体を301リダイレクトでtokaiair.comに転送。

| 項目 | 評価 |
|------|------|
| メリット | 設定5分で完了。管理コストゼロ。ドメイン更新費のみ。タイプミスや類似ドメインからの流入を拾える |
| デメリット | ドメインの積極活用とは言えない。将来の柔軟性を残さない |
| SEO影響 | 301リダイレクトなので微プラス（被リンクがあれば本体に集約） |
| 管理コスト | **ゼロ**（DNS設定のみ） |
| 初期構築 | 5分 |

## 推奨: 案4（即時） → 案3（短期） → 案1（広告出稿時）

### フェーズ1: 今すぐ（案4）
**301リダイレクト設定のみ実施。**
- 工数: 5分
- 理由: ドメインを放置するとスパムサイトに悪用されるリスクがある。最低限のリダイレクト設定で防御
- コスト: ドメイン更新費のみ

### フェーズ2: 余裕ができたら（案3）
**短縮URLサービスとして活用。**
- 工数: 数時間
- 実装案: Cloudflare Workers（無料枠）でパスベースのリダイレクト
  - `短縮ドメイン/drone` → `tokaiair.com/drone-survey/`
  - `短縮ドメイン/quote` → `tokaiair.com/contact/`
  - `短縮ドメイン/case/xxx` → 各事例ページ
- 効果: 名刺・提案書・メール署名で使用。クリック計測で営業効果を可視化
- 特に新美・政木の営業活動でどの資料が参照されているかデータ取得可能

### フェーズ3: 広告予算確保後（案1）
**広告LP専用として活用。**
- 前提: 広告出稿の予算と計画が確定してから着手
- 実装: 静的HTML（Cloudflare Pages無料枠）でLP作成
- 現時点では着手不要

### 案2（マイクロサイト）は非推奨
- ドメインパワーの分散はSEO上の明確なマイナス
- 特にBing 68.6%の流入構成を考えると、Bingはドメインオーソリティを重視するため分散は致命的
- 44本の記事資産を活かすにはtokaiair.comへの一極集中が正解

---

## 注意事項

- **案2は絶対に避ける**: 現在のSEO資産（記事44本+15本、Bing 68.6%流入）を毀損するリスクが高すぎる
- **ドメイン更新は忘れずに**: 未使用でも失効させるとスパムサイトに取得される
- **社外秘情報の掲載禁止**: どの案を採用しても、顧客名・営業成績・ランウェイ等は絶対に掲載しない
