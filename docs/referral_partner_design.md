# 紹介パートナー制度設計書

作成日: 2026-03-15
ステータス: 設計完了・実装待ち

## 概要

TOMOSHI事業（AI業務改善コンサルティング）の紹介パートナー制度。
税理士・社労士・中小企業診断士・ITベンダー等からの紹介経由で案件を獲得する仕組み。

## 紹介手数料体系

| 紹介パターン | 手数料率 | 支払条件 |
|-------------|---------|---------|
| 商談設定のみ（アポ紹介） | 初回受注額の10% | 入金後30日以内 |
| 同席・クロージング支援 | 初回受注額の15% | 入金後30日以内 |
| 継続案件（月額契約） | 月額の10% x 6ヶ月 | 毎月末締め翌月払い |

### 上限・最低保証
- 1件あたり手数料上限: 50万円
- 最低手数料: 1万円（少額案件でも紹介者に還元）
- 紹介から90日以内の受注が対象

## 紹介フロー

```
パートナー → 紹介連絡（Lark/メール）
  ↓
TAS → CRM「紹介元」フィールドに記録
  ↓
初回面談実施（TOMOSHI担当）
  ↓
受注 → 請求・入金確認
  ↓
手数料計算 → パートナーへ支払通知 → freee経由で振込
```

## パートナー契約テンプレート（要素）

1. **契約形態**: 業務委託（紹介業務）
2. **契約期間**: 1年（自動更新）
3. **秘密保持**: 顧客情報の取り扱い規定
4. **反社条項**: 標準反社排除条項
5. **紹介方法**: メール/Lark/電話での事前連絡必須
6. **禁止事項**: 直接営業の禁止、競合紹介の制限なし
7. **手数料支払**: 上記体系に準拠、請求書ベース
8. **解約**: 30日前通知、既発生手数料は支払義務あり

## CRM管理方法

### Lark Base 商談テーブル追加フィールド
- `紹介元パートナー`: リンクフィールド（→ 取引先テーブル）
- `紹介手数料ステータス`: 単一選択（未計算/計算済み/支払済み）
- `紹介手数料額`: 数値

### Lark Base 取引先テーブル
- 既存の取引先テーブルにパートナーも登録
- `取引先種別` フィールドで「紹介パートナー」を選択肢に追加

### 自動化（将来）
- 受注確定時に紹介手数料を自動計算（GitHub Actions）
- 月次で未払い手数料レポートを生成
