## 5. Spark DBの活用提案

### 157,681通のメールデータは以下に活用可能:

1. **過去案件の完全復元**: 見積→受注→納品→請求の流れを時系列で再構築
2. **顧客別取引履歴**: 188社の取引頻度・金額・案件種別を分析
3. **メールテンプレート最適化**: 過去の成約メールパターンから最適な文面を学習
4. **営業パイプライン分析**: 見積→受注の転換率を顧客別・案件種別で算出
5. **価格戦略**: 過去見積金額の傾向分析（メール本文に金額記載があれば）

### 技術的注意点
- Spark Desktop SQLiteは読み取り専用で利用（WALファイルあり、アプリ稼働中はコピーして使用）
- `shortBody`フィールドにメール冒頭テキストあり（全文は別途取得が必要な場合あり）
- `messageAttachment`テーブルに添付ファイル情報あり（見積書PDFの特定に使用可能）
