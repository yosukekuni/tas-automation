# TOMOSHI 公的機関登録手順

作成日: 2026-03-14 / ステータス: 凍結（設計書のみ）

## 概要

TOMOSHI事業部の信頼性向上と補助金ビジネスへの参入のため、経営革新等支援機関の認定とIT導入補助金のIT導入支援事業者登録を行う。

---

## 1. 経営革新等支援機関の認定申請

### 概要

中小企業庁が認定する「認定経営革新等支援機関」になることで、補助金申請支援・経営相談等の公的支援業務を行える。

### 申請要件

- 税務、金融、企業財務に関する専門的知識を有すること
- 中小企業支援の実務経験（原則3年以上、または同等の能力）
- 法人または個人事業主であること

### 申請手順

| Step | 内容 | 期間 |
|------|------|------|
| 1 | 中小企業庁「認定経営革新等支援機関」ページで要件確認 | 1日 |
| 2 | 電子申請システム（e-Gov or 専用サイト）でアカウント作成 | 1日 |
| 3 | 申請書類作成（実務経験証明、専門知識証明） | 1〜2週間 |
| 4 | 申請提出 | 1日 |
| 5 | 審査期間 | 1〜3ヶ月 |
| 6 | 認定通知 | - |

### 必要書類

- 申請書（所定様式）
- 実務経験を証明する書類（支援実績一覧、契約書等）
- 専門知識を証明する書類（資格証明、研修修了証等）
- 登記簿謄本（法人の場合）
- 財務諸表（直近2期分）

### 費用

- 申請手数料: 無料
- 書類準備の間接コスト: 10〜20時間

### 注意点

- AI活用コンサルティングのみの実績では要件を満たさない可能性
- 東海エアサービス本体（ドローン測量）の経営実績を含めて申請するのが現実的
- 認定後は5年ごとの更新が必要

## 2. IT導入補助金 IT導入支援事業者登録

### 概要

経済産業省のIT導入補助金制度において、ITツールを提供する事業者として登録する。顧客企業がTOMOSHIのサービスを補助金で導入できるようになる。

### 申請要件

- 日本国内で法人登記されていること
- 安定的な事業基盤を有すること
- 情報セキュリティに関する取り組み（SECURITY ACTION二つ星宣言）
- 補助事業を適正に遂行できる体制

### 申請手順

| Step | 内容 | 期間 |
|------|------|------|
| 1 | SECURITY ACTION二つ星を宣言（IPA公式サイト） | 1〜2週間 |
| 2 | gBizIDプライムの取得（未取得の場合） | 2〜3週間 |
| 3 | IT導入補助金公式サイトで支援事業者登録申請 | 1日 |
| 4 | ITツールの登録（TOMOSHIのサービスを登録） | 1週間 |
| 5 | 審査期間 | 2〜4週間 |
| 6 | 登録完了 | - |

### 必要書類・準備

- gBizIDプライム（法人代表者のマイナンバーカード必要）
- SECURITY ACTION二つ星宣言番号
- 登記簿謄本
- 直近の決算書
- ITツール情報（サービス内容、価格、導入効果等）
- コンソーシアム構成（単独 or 複数社連携）

### 費用

- 登録手数料: 無料
- SECURITY ACTION宣言: 無料
- gBizID取得: 無料
- 間接コスト: 20〜30時間

### 登録するITツール案

| ツール名 | カテゴリ | 補助対象額（例） |
|---------|---------|----------------|
| AI業務自動化パッケージ | 業務プロセス | 50〜150万円 |
| DX推進コンサルティング | コンサルティング | 30〜100万円 |
| AI導入診断+PoC | 分析ツール | 20〜50万円 |

---

## 全体スケジュール

```
経営革新等支援機関: 準備2週間 + 審査1〜3ヶ月 = 最短2ヶ月
IT導入支援事業者: 準備3〜4週間 + 審査2〜4週間 = 最短2ヶ月
→ 両方並行で進めれば 最短2〜3ヶ月 で完了
```

## 工数見積もり

| 作業 | 工数 |
|------|------|
| 要件調査・確認 | 4時間 |
| SECURITY ACTION宣言 | 2時間 |
| gBizID取得（未取得の場合） | 2時間 + 待ち |
| 申請書類作成（支援機関） | 10時間 |
| ITツール登録資料作成 | 8時間 |
| **合計** | **26時間 + 審査待ち** |

## 優先順位

**低**: TOMOSHI事業部の実績蓄積が先。補助金ビジネスは実績3件以上になってから着手が現実的。ただしgBizIDプライムとSECURITY ACTIONは先に取得しておくと後が楽。
