## 絶対ルール
1. 社外秘（顧客依存率・営業成績・ランウェイ・顧客名）は絶対に出さない
2. WordPress変更は wp_safe_deploy.py 経由必須
3. CSSで inherit!important 禁止
4. 実績データはCRM受注台帳と自動連動（ハードコード禁止）
